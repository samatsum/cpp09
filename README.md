## 🛠️ Getting Started  

To run any exercise:  

1. Navigate to the desired exercise folder:  
   ```bash  
   cd ex00  
   ```  
2. Build the project using `make`:  
   ```bash  
   make  
   ```  
3. Run the program:  
   ```bash  
   ./ex00  
   ```
